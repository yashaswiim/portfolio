This project is called "Let's Decide" since it is about the user making an important decision. Essentially, a group of 4 people cannot decide on the art-style for their comic and the user is able to follow them "into" their mind - a forest - where they reach a crossroad and have to choose between two different art-styles. "Let's Decide" follows the conventions of a comic book, however adapted for the web, like having each panel adapt to the widescreen size of a webpage instead of gutters to separate. At the end of each path when the user makes the choice, the final panels both prompt the user to start again! Basically, this leads the user to try the other path no matter which path they choose initially. This project was designed as a funny way of looking at how difficult it is to make simple decisions, and we just go round and round...
